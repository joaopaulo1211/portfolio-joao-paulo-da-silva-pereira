# SM7 - Engenharia de Software: Aplicativo de Delivery (Bubble.io)

> **Status:** Concluído ✅
> **Tecnologia:** No-code, Bubble.io, API Integration, Engenharia de Software.

---

## 📝 Descrição do Projeto

Este projeto consiste no desenvolvimento de um **Aplicativo de Delivery** completo utilizando a plataforma *no-code* **Bubble.io**. O objetivo foi aplicar conceitos de engenharia de software, design de interface (UI) e experiência do usuário (UX) em uma solução funcional e escalável para o mercado de logística e alimentação.

O aplicativo gerencia desde a visualização de produtos e busca de estabelecimentos até o fluxo de checkout e acompanhamento de pedidos.

---

## 🚀 Funcionalidades Principais

* **Busca Inteligente:** Sistema de filtros e pesquisa por categorias e nomes de produtos/lojas.
* **Carrinho de Compras:** Gestão dinâmica de itens, quantidades e cálculos de subtotal.
* **Gestão de Pedidos:** Fluxo completo desde a emissão até o status final.
* **Interface Responsiva:** Otimizada para dispositivos móveis (Mobile-first).
* **Integração de Dados:** Banco de dados relacional configurado para gerenciar usuários, produtos e pedidos em tempo real.

---

## 🛠️ Tecnologias e Ferramentas

* **Bubble.io:** Engine principal para desenvolvimento visual e lógica de workflows.
* **API Connector:** Configuração para futuras integrações de pagamento ou geolocalização.
* **Bubble Database:** Modelagem de dados estruturada para performance.

---

## 📐 Conceitos Aplicados

* **Arquitetura de Dados:** Criação de tipos de dados (Data Types) e relacionamentos entre tabelas.
* **Workflows Logics:** Programação visual de eventos (ao clicar em "comprar", adicionar ao carrinho).
* **Design System:** Manutenção de padrões visuais para garantir uma navegação intuitiva.
